# IonChem.jl

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.18631983.svg)](https://doi.org/10.5281/zenodo.18631983)


IONCHEM is a dynamic ion chemistry model for the E-region during auroral precipitation.
It integrates the auroral production and chemical reactions to calculate time-dependent ion densities.
